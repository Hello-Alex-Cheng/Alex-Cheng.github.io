环境

  • node: v12.14.0
  • hexo 1.1.9

启动

  • hexo s

创建文章

  • hexo new <文章标题>

表示要创建的类型，比如 post，draft, about

创建草稿文章

  • hexo new draft demo

查看草稿

  • hexo s –draft

生成静态资源

  • hexo g

发布

  • hexo d
